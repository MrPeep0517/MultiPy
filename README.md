## Commands:

- ```Now```:
  - Prints out the current date and current time.
- ```Calc```
  - Asks for 3 things; first number, operation (can be; +, -, *, /, f (factorial), or ^), and second number.
- ```Quit/Exit```
  - Stops the program.
- ```Py```
  - Runs the python console. Type ```exit()``` to return.
